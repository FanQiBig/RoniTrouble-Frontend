# 使用 uni-app 默认导航栏

## 修改文件
`pages.json`

## 具体改动

将 chat 页面的 `"navigationStyle": "custom"` 删除，使用默认导航栏

从：
```json
{
    "path": "pages/chat/chat",
    "style": {
        "navigationStyle": "custom"
    }
}
```

改为：
```json
{
    "path": "pages/chat/chat"
}
```

## 注意事项

* chat.vue 已经有 `uni.setNavigationBarTitle({ title: name.value })` 调用，会自动设置导航栏标题为聊天对象名称
* 默认导航栏会自动提供返回按钮
