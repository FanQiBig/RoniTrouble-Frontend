# 修正 profile.js API 路径以匹配 openapi.yaml

## 修改文件
`api/profile.js`

## 具体改动

### 1. 修改 API 路径
- `isProfileExisted()` - 将 `/profile` 改为 `/userInfo/isExisted`
- `getMyProfile()` - 将 `/profile/me` 改为 `/userInfo/me`
- `saveProfile(data)` - 将 `/profile` 改为 `/userInfo`
- `getProfileStats()` - 删除此函数（openapi.yaml 中没有对应接口）

### 2. 添加缺失的 API 函数
- `getUserPosts(userId)` - 获取用户帖子 (`/userInfo/posts`)
- `getUserComments(userId)` - 获取用户评论 (`/userInfo/comments`)
- `getUserLikedPosts(userId)` - 获取用户点赞的帖子 (`/userInfo/likedPosts`)
- `getUserPostCount(userId)` - 获取用户帖子数 (`/userInfo/postCount`)
- `getUserViewCount(userId)` - 获取用户浏览数 (`/userInfo/viewCount`)
- `getUserLikeCount(userId)` - 获取用户点赞数 (`/userInfo/likeCount`)
