# 添加 IP 获取和上传功能

## 需要创建/修改的文件

### 1. 创建 utils/ipService.js

* 从 Robabuny-Frontend 复制 ipService.js

* 获取用户公网 IP 地址

### 2. 修改 api/auth.js

* 添加 reportIP 函数，调用 POST /ip 接口

* 参数：ip (string), userId (integer, optional)

### 3. 修改 App.vue

* 导入 getIPAddress 和 reportIP

* 在 onLaunch 时调用 reportUserIP 函数

* reportUserIP 函数会获取 IP 并上传到服务器
