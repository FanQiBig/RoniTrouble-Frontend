# 复制图标文件到新项目

## 需要复制的文件

从 `Robabuny-Frontend/icon/` 复制到 `icon/` 目录：
- iconfont.css - 图标样式文件
- iconfont.js - 图标 JavaScript 文件
- iconfont.json - 图标 JSON 配置
- iconfont.ttf - 图标字体文件
- demo.css - 演示样式（可选）
- demo_index.html - 演示页面（可选）

## 需要修改的文件

**App.vue** - 添加 iconfont.css 导入
```css
@import './icon/iconfont.css';
```
